# LeaveFlow — Leave Management System
**Module:** SD-WM01 
**Organisation:** Chacks Creative Technologies
**Date:** June 2026

---

## Project Overview
LeaveFlow is a web-based Leave Management System that allows employees 
to submit leave requests and enables HR managers to approve or reject 
requests, manage user accounts, and maintain leave records.
